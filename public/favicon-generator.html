Favicon 生成器

当前状态

您的网站现在使用 /images/logo.png 作为favicon。

如果浏览器标签页没有显示图标，请尝试以下步骤：

Favicon 预览

16x16 像素 (标准favicon尺寸)
32x32 像素 (高清favicon尺寸)

故障排除步骤

  1. 强制刷新浏览器: 按 Ctrl+F5 (Windows) 或 Cmd+Shift+R (Mac)
  2. 清除浏览器缓存: 清除浏览器缓存和Cookie
  3. 检查文件路径: 确认 /public/images/logo.png 文件存在
  4. 重启开发服务器: 停止并重新启动 npm run dev
  5. 检查浏览器控制台: 查看是否有404错误

测试链接

点击以下链接测试favicon是否正确加载：

  • 直接访问logo.png
  • 访问首页

如果仍然不显示

如果favicon仍然不显示，可能需要：

  • 将logo.png转换为favicon.ico格式
  • 使用在线favicon生成器
  • 检查logo.png文件是否损坏
